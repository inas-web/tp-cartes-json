# TP React Native - Catalogue de Cartes

Application mobile réalisée avec **React Native** et **Expo**.  
Ce projet affiche un catalogue de cartes à partir d'un fichier JSON.

## 🚀 Fonctionnalités
- Liste de cartes (image, titre, description)
- Clic sur une carte pour ouvrir un lien web
- Design moderne et responsive

## 🧩 Structure
- `App.js` → point d'entrée
- `components/` → composants réutilisables
- `data/cards.json` → données locales
